# Algorithms And Complexity

Этот подпакет не про олимпиадщину, а про базу, которую часто спрашивают на backend собеседованиях.

Материалы:
- [Time And Space Complexity](./time-and-space-complexity.md)
- [Common Algorithm Patterns And Examples In Go](./common-algorithm-patterns-and-examples-in-go.md)

Что важно уметь:
- объяснить `O(1)`, `O(log n)`, `O(n)`, `O(n log n)`, `O(n^2)`;
- понять, где алгоритмическая сложность бьет по production path;
- назвать типовые структуры и алгоритмы без лишней теории;
- показать примеры на слайсах, мапах, строках и сортировке в Go.
