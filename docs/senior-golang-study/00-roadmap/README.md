# Roadmap

Здесь держи общий план подготовки и прогресс.

Что полезно собрать:
- чек-лист тем, которые уже закрыты;
- список слабых мест;
- план по неделям;
- перечень company-specific тем, если готовишься под конкретные вакансии.

Минимальный senior baseline:
- уверенно объяснять runtime Go и конкурентность;
- проектировать сервис с учетом SLA, отказов и observability;
- разбираться в SQL, индексах, транзакциях и Redis;
- понимать Kafka, RabbitMQ или NATS хотя бы на уровне trade-offs;
- уметь аргументировать архитектурные решения, а не только писать код.
