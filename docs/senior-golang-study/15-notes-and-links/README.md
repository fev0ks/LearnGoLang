# Notes And Links

Это служебный раздел для быстрого накопления материала.

Сюда можно складывать:
- сырые заметки после интервью;
- ссылки на статьи, видео, доклады;
- backlog тем, которые потом нужно разложить по разделам;
- тезисы по вакансиям разных компаний.

Рекомендуемый подход:
- сначала быстро фиксировать мысль здесь;
- потом переносить заметки в профильный раздел;
- оставлять только то, что еще не разобрано.

## Что сюда складывать

- ссылки на статьи с короткой пометкой "зачем читать";
- неожиданные вопросы с интервью;
- спорные trade-offs, которые хочется позже разобрать глубже;
- краткие выводы после каждой lab или mock interview.

## Вопросы

- что из накопленных заметок уже пора разложить по профильным разделам;
- какие ссылки ты сохраняешь "на потом" без реальной причины;
- какие 5 тем повторяются чаще всего и требуют отдельного конспекта;
- можешь ли ты по своим заметкам быстро восстановить пробелы в знаниях.
