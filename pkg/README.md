# Папка `pkg`

Это не production `pkg` в смысле "публичное API модуля", а учебная коллекция.

Разделы:
- `code_examples` — теперь в основном редирект на новые тематические каталоги в `topics/`;
- исторический `everything` уже вынесен в `topics/07-mixed-sandbox/everything`.
- `funcs`, `types`, `logging`, `web`, `benchmarks` — оставшиеся служебные и переиспользуемые пакеты.
- `pkg/pkg/logging` — дублирующий старый пакет, который лучше потом либо удалить, либо слить с основным `logging`.

Практический ориентир:
- если хочется быстро вспомнить конкретную ловушку Go, смотри `topics/01`, `topics/02` и `topics/03`;
- если хочется потренироваться на задаче, смотри тематические каталоги в `topics/`.
