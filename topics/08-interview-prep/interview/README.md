# Interview

Здесь материалы лучше воспринимать в трех слоях:
- `before/` — заметки и теория до решения;
- `after/` и `after_gpt/` — варианты решений и разборов после;
- `questions.txt` — быстрый список вопросов для самопроверки.

Полезный формат для новых записей:
- вопрос;
- короткий ответ своими словами;
- trade-offs;
- пример из практики или маленький кодовый sketch.
