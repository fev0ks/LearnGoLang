# Mixed Sandbox

Это остаточная зона, куда попали старые и смешанные эксперименты.

Основные каталоги:
- `topics/07-mixed-sandbox/cmd`
- `topics/07-mixed-sandbox/cmd2`
- `topics/07-mixed-sandbox/cmd3`
- `topics/07-mixed-sandbox/everything`
- `topics/07-mixed-sandbox/image_byte`

Как с этим работать:
- использовать как временный sandbox можно, но лучше не разрастать;
- если пример стал полезным, переносить его в профильную тему;
- `everything` постепенно разбирать по другим разделам, а не складывать туда новое.

Из этой зоны уже вынесено все, что выглядело как стабильный учебный материал.
